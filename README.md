# GrindIT — Your GitHub Story, Unwrapped

> All Rights Reserved. Proprietary and confidential.

![GrindIT](./public/preview.png)

---

GrindIT turns your GitHub profile into a cinematic developer recap — like Spotify Wrapped, but for coders.

**How it works:**

1. Enter your GitHub username
2. Pick a time period and an AI tone (Funny, Brutal or Motivational)
3. Hit Generate
4. Get your story as animated slides — archetype, commits, languages, top repo, achievements and more
5. Save or share your card to X or LinkedIn
